# 周计划-html
一页上完整浏览一周的时间安排

FlowWeek 使用说明书
欢迎使用 FlowWeek。这是一款融合了极简美学与强大逻辑的周计划工具，致力于通过自然的交互方式，帮助您从混乱的日程中找回秩序与专注。
1. 快速开始 (Getting Started)
界面布局
应用分为左右两个主要区域，中间由分割线隔开：
左侧（专注面板）：显示当前时间、日期、正在进行的任务（Current Focus）以及今日任务清单（Today's Plan）。
右侧（周视图）：您的主要规划区域，显示一周的时间网格。
调整布局
拖动调整：将鼠标悬停在左右面板的分界线上，按住鼠标左键左右拖动，可自由调整侧边栏与表格的宽度比例。
一键归位：点击左侧工具栏的 重置图标 (⟲)，可瞬间恢复默认宽度、回到浅色模式并跳转至本周。
2. 任务管理 (Task Management)
创建任务
FlowWeek 提供两种创建方式：
拖拽创建（推荐）：在右侧时间格的任意空白处，按住鼠标左键向下拖动。拖动过程中，选框上下会实时显示“开始时间”和“结束时间”（自动吸附15分钟刻度）。松开鼠标即可弹出设置窗口。
加号按钮：点击界面右下角的黑色悬浮按钮 (+)，手动输入时间创建任务。
编辑任务
单击编辑：点击任意已存在的色块即可修改内容。
重叠选择：如果同一时间段有多个重叠任务，点击时会自动弹出一个微型菜单，列出所有重叠任务，请从中选择您想编辑的那一个。
任务详情设置
在弹窗中，您可以详细配置：
标题与颜色：输入任务名，并从 9 种莫兰迪/高亮色系中选择代表色。
时间与次日：设置开始和结束时间。如果任务跨越午夜（例如 23:00 到 次日 01:00），请勾选 "Next Day" (次日)，系统会自动处理跨天显示。
路程 (Commute)：拖动滑块设置通勤时间。表格中会在任务开始前显示一段虚线区域，代表路程时间。若路程跨越了前一天的午夜，也会自动显示在前一天的底部。
重复设置：
Weekly (每周)：勾选后，任务每周循环。可设置“重复 N 周后停止”。
Daily (每天)：勾选后，任务将在本周及未来的每一天显示（自动包含每周重复属性）。
3. 视图与导航 (Navigation)
切换周次：将鼠标移动到右侧表格的顶部日期栏区域，左上角和右上角会浮现圆形箭头按钮。点击即可切换“上一周”或“下一周”。
回到本周：当您浏览过去或未来的周次时，顶部中央会出现 "Back to Today" 按钮，点击即可快速返回当前周。
实时红线：表格中有一条横穿的红色实线，精准指示当前的实时时间。
4. 专注面板 (Focus Sidebar)
左侧面板是您的“当下”仪表盘：
Current Focus (当前专注)：系统会根据当前时间自动锁定正在进行的任务，并显示动态进度条和剩余时间。
Today's Plan (今日计划)：底部区域按时间顺序列出今天的所有任务。
灰色删除线：已结束的任务。
高亮加粗：正在进行的任务。
普通灰色：尚未开始的任务。
5. 个性化与设置 (Settings)
位于左侧边栏顶部的时间下方：
语言切换：点击 EN | 中 | 日 可实时切换界面语言。
暗色模式：点击 太阳/月亮 图标切换主题。暗色模式下，任务颜色会自动调整为更醒目的荧光粉彩系，确保可视性。
全部删除：点击 垃圾桶 图标可清空所有数据（注意：此操作会弹出二次确认框，一旦确认无法恢复）。
常见问题 (FAQ)
Q: 为什么我的任务在表格里显示了两段？
A: 如果您设置了“次日结束”或路程时间跨越了午夜（00:00），系统会自动将其分割：前半段显示在当天的底部，后半段显示在次日的顶部，以符合真实的时间逻辑。
Q: 刷新页面后数据还在吗？
A: 在。所有数据（包括任务、侧边栏宽度、语言设置、暗色模式状态）都安全地存储在您的本地浏览器中。
Q: 如何删除一个任务？
A: 点击该任务色块，在弹出的窗口右下角点击 "Delete" 按钮即可。
